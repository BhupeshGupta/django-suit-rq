django-suit-rq
==============

Support the django-rq admin when using django-suit

Quick Start
-----------

1. Install the package from pypi:

    .. code-block:: bash

        pip install django-suit-rq

2. Add "suit_rq" your INSTALLED_APPS:

    .. code-block:: python

        INSTALLED_APPS = (
            'suit',
            'suit_rq',
        )
